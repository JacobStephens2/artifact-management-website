Details about use by and play by dates

The use by date for determined by the two following rules.

  • If the object has not yet been used, the use by date is 90 days from the acquisition date.
  • Otherwise, the use by date is 180 days from the most recent use date.

The list is limited to the 100 nearest use by date objects. Keeping these rules enables kept objects to be used either within the past 90 days or the coming 90 days. This approach is probably best used as a guideline rather than as a strict rule. The Minimalists' 90/90 Rule inspired Jacob, operator of Steward Goods, to use these rules to track their possessions.
